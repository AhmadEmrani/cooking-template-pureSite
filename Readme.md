# 🍽️ Cooking Template – PureSite

A modern, responsive, and lightweight cooking website template designed for food blogs, recipe websites, culinary portfolios, and restaurant-related landing pages. Built with simplicity and maintainability in mind, this static template provides a clean user experience while remaining easy to customize and deploy.

---

## 📖 Overview

**Cooking Template – PureSite** is a fully responsive static website template created for showcasing recipes and food-related content in an elegant and user-friendly way.

Whether you're a food blogger, a cooking enthusiast, a frontend developer looking for a UI reference, or someone building a recipe website, this template provides a solid foundation to get started quickly.

---

## ✨ Features

- Modern and clean user interface
- Fully responsive design
- Lightweight and fast-loading
- Easy to customize and extend
- Well-organized project structure
- Beginner-friendly codebase

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/AhmadEmrani/cooking-template-pureSite.git
```

### Navigate to the Project Directory

```bash
cd cooking-template-pureSite
```

### Run the Project

Since this is a static website, no dependencies or build tools are required.

Simply open the `index.html` file in your preferred browser.

---

## 💻 Running Locally

### Option 1: Open Directly

Double-click the `index.html` file to launch the website in your browser.

---

### Option 2: Using VS Code Live Server

For a better development experience:

1. Open the project in Visual Studio Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html`.
4. Select **Open with Live Server**.

The website will automatically reload whenever changes are saved.

## 🌐 Deployment

You can deploy this template on any static hosting platform.

### GitHub Pages

1. Push your project to GitHub.
2. Go to:

```
Settings → Pages
```

3. Select the branch you want to publish.
4. Save your changes.
5. Your website will be available through GitHub Pages.
